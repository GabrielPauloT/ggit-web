1. O Prompt da IA (O que é digitado rápido no topo)

> prompt: Crie um componente de card para rastreamento de frota logística em tempo real. Use React, Tailwind CSS e ícones do Lucide. O status deve ser dinâmico.
2. O Código Real (Lado Esquerdo - Next.js/React + Tailwind)
Este é um código limpo, moderno, que usa o padrão atual da indústria (como os componentes do shadcn/ui). Se um dev ler isso, ele vai respeitar na hora.

TypeScript
import { Badge } from "@/components/ui/badge";
import { MapPin, Truck } from "lucide-react";

interface RouteProps {
  id: string;
  origin: string;
  destination: string;
  status: 'in_transit' | 'pending' | 'delivered';
  driver: string;
}

export default function ActiveRouteCard({ id, origin, destination, status, driver }: RouteProps) {
  return (
    <div className="p-6 bg-white dark:bg-zinc-900 rounded-xl border border-zinc-200 shadow-sm">
      <div className="flex justify-between items-center mb-4">
        <h3 className="text-lg font-semibold text-zinc-900 dark:text-white">
          Rota #{id}
        </h3>
        <Badge variant={status === 'in_transit' ? 'default' : 'secondary'}>
          {status === 'in_transit' ? 'Em Trânsito' : 'Pendente'}
        </Badge>
      </div>
      
      <div className="flex items-center space-x-4 mb-6 text-sm text-zinc-500">
        <div className="flex items-center">
          <MapPin className="w-4 h-4 mr-1 text-blue-500" />
          <span>{origin}</span>
        </div>
        <div className="h-px w-8 bg-zinc-300 dark:bg-zinc-700"></div>
        <div className="flex items-center">
          <MapPin className="w-4 h-4 mr-1 text-emerald-500" />
          <span>{destination}</span>
        </div>
      </div>

      <div className="flex items-center space-x-3 bg-zinc-50 dark:bg-zinc-800/50 p-3 rounded-lg">
        <Truck className="w-5 h-5 text-zinc-700 dark:text-zinc-300" />
        <div>
          <p className="text-sm font-medium text-zinc-900 dark:text-white">{driver}</p>
          <p className="text-xs text-emerald-500 font-medium">Chegada estimada: 2h 15m</p>
        </div>
      </div>
    </div>
  );
}
3. A Interface que Aparece (Lado Direito)
Enquanto esse código é "digitado" na esquerda, a tela da direita renderiza exatamente o que o código descreve:

Um Card minimalista com bordas arredondadas e sombra suave (estilo Vercel/Stripe).

No topo: O título "Rota #BRZ-8492" e do lado direito uma tag (Badge) preta ou azul escrito "Em Trânsito".

No meio: Um ícone de pino de mapa azul escrito, por exemplo, "Melbourne, VIC", uma linha cinza conectando a outro pino verde escrito "Sydney, NSW".

Embaixo: Um bloco com fundo levemente acinzentado destacando um ícone de caminhão, o nome do motorista (ex: "Carlos Silva") e o texto em verde "Chegada estimada: 2h 15m".

Por que esse é o cenário perfeito?
É técnico na medida certa: Mostra tipagem no TypeScript (interface RouteProps), uso de utilitários modernos (Tailwind) e componentização correta.

Mostra valor de negócio: Rastreamento de frota e gestão de rotas são dores caríssimas para empresas de logística. Você prova que sabe construir a solução.
